CSS Subgrid

"Grids all the way down!"

CSS 📋 
/* Parent grid */
.product-grid {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 1rem;
}

/* Child inherits parent grid */
.product-card {
    display: grid;
    grid-template-rows: subgrid;
    grid-row: span 4;
    
    /* Child elements align across all cards */
    & .image { grid-row: 1; }
    & .title { grid-row: 2; }
    & .description { grid-row: 3; }
    & .price { grid-row: 4; }
}
HTML 📋 
<div class="product-grid">
    <div class="product-card">
        <img class="image">
        <h3 class="title">Product Name</h3>
        <p class="description">Description</p>
        <span class="price">$99</span>
    </div>
</div>
