CSS :has() - The Parent Selector

"CSS can finally look up the tree"

/* CSS :has() Selector */ form:has(input:invalid) { background: rgba(255, 71, 87, 0.1); border: 2px solid #ff4757; padding: 1rem; border-radius: 10px; } input:invalid { border-color: #ff4757; } /* HTML */ <form> <input type="email" required> <input type="tel" pattern="[0-9]+"> </form>
